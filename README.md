# 🧠 EchoBrainz — Mood Tracking & Community Engagement Platform

EchoBrainz is a full-stack web application that allows users to track mood patterns, maintain journals, and engage with a supportive community — complete with interactive dashboards to visualize personal activity over time.

---

## 🚀 Key Features

- 📝 Mood tracking and journaling workflows
- 👥 Community interaction system
- 🔄 Real-time data synchronization
- 📊 Personal activity dashboard
- 🔔 Reminder notification system

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | React.js, TypeScript |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB |

---

## 🗂️ Project Structure

```
frontend/    → React UI components
backend/     → API routes and business logic
database/    → MongoDB schemas
```
